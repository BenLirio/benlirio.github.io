        Data Conversion

        I came across this use case of ChatGPT while working on this project. I aimed to create a citation for an article in CSL-JSON format. I had a long list of authors, but the formatting needed to be fixed when I copied and pasted them. So I asked ChatGPT the following to help me with citations.

        Analysis

        I could then use the response from ChatGPT in my project without having to do the manual work of parsing each name individually.

        This example raises an important question. What if ChatGPT is wrong? I want to counter that question with “What if I am wrong?” or “What if my computer glitches and types the wrong letter?” or “What if < blank > ?”

        Hidden in my first question about ChatGPT’s correctness is an implicit assumption that ChatGPT is the only point of failure. However, a better way of thinking about this is “Who is responsible?” If I trust ChatGPT to give me the correct answer and it produces something bogus that I do not verify, I am responsible, not the chatbot.

        This is why education around the limitations of GPT is essential so that people can better take responsibility for the output.
